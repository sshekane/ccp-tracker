Sharing workflow
1. One person clicks Save to File in the header → downloads a .json file
2. Share that .json (email, Teams, shared drive) with a colleague
3. Colleague opens the HTML file, clicks Load from File in the header → picks the .json
4. Their copy now has your latest data. They make changes, save again, share back.
